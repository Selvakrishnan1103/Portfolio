Connecting Frontend with Python Flask Backend


Introduction


In this blog, I will walk you through a simple yet effective way to connect your frontend (HTML, CSS) with a Python Flask backend without the need for JavaScript. We’ll create a basic form where users can submit their name and email, and the backend will process the form data and display a response on a new page.


Step 1: Frontend (HTML, CSS)


index.html


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Form Submission</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <div class="container">
        <h1>Submit Your Information</h1>
        <form action="/submit" method="POST">

            <label for="email">Email:</label>
            <input type="email" id="email" name="email" required>
            
            <button type="submit">Submit</button>

</body>
</html>

Step 2: Backend (Python Flask)


app.py


from flask import Flask, request, render_template, redirect, url_for

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/submit', methods=['POST'])
def submit_form():
    name = request.form['name']
    email = request.form['email']

    # Simulate processing the data
    response_message = f"Hello, {name}! Your email {email} has been submitted successfully."

    return render_template('result.html', message=response_message)

if __name__ == '__main__':
    app.run(debug=True)

result.html

<!DOCTYPE html>
<html lang="en">

<body>
    <div class="container">
        <h1>Submission Result</h1>
        <p>{{ message }}</p>
        <a href="{{ url_for('home') }}">Go Back</a>
    </div>
</body>
</html>

Step 3: Running the Application


Set Up the Project Directory:


Create a directory for your project. Inside this directory, create three subdirectories: static, templates, and static/css.
Place index.html and result.html inside the templates folder. Place styles.css inside the static/css folder.


Install Flask:


If you haven’t installed Flask yet, run:


pip install Flask

Run the Flask Application:


In your terminal, navigate to the directory containing app.py and run:


python app.py

This will start the Flask development server. Open your web browser and go to http://127.0.0.1:5000/ to see your form.


Test the Form:


Fill out the form and submit it. You should see a new page displaying the submission result.


Explanation:


In this example, the frontend consists of a simple form built with HTML and styled with CSS. The form submits the user's name and email to the Flask backend via a POST request. The backend processes this data and then renders a new HTML page (result.html) that displays a message confirming the submission.


This method demonstrates a straightforward way to handle form submissions in web applications without relying on JavaScript.
