GPA Calculator

Project Overview


This project is a GPA calculator developed using HTML, CSS, JavaScript, and Java. It was later converted into an Android app using Android Studio. The app allows users to calculate their GPA and track academic performance efficiently.


Technologies used


HTML, CSS, JavaScript, Java, Android Studio


Features


It is designed with a clean and intuitive interface that allows users to input their grades and credits effortlessly.

It automatically calculates the GPA as the user inputs their grades, providing immediate feedback on their academic performance.

It supports the calculation of GPA across multiple semesters, allowing users to track their academic progress over time.

It accepts Letter grades and converts them to the GPA scale.

It includes validation checks to ensure that inputs are valid and within acceptable ranges, minimizing user errors.

The app is responsive and adapts to different screen sizes, ensuring usability on both desktop and mobile devices.

Student Information System (SIS)

Project Overview


The Student Information System (SIS) is a comprehensive platform developed using Python Flask, MySQL, and HTML/CSS. It is designed to manage student data and academic records efficiently. The system allows educational institutions to maintain detailed records of student information, academic performance, and other essential data.


Technologies used


Python Flask

MySQL

HTML

CSS


Features


It efficiently manages student information, including personal details, contact information, and academic records.

It implements a secure authentication system with different user roles (e.g., admin, teacher, student) to control access to various features.

It keeps track of student attendance, with options to mark presence, absence, and leave. Generates attendance reports for review.

It is built with a scalable architecture that allows for the addition of new features and modules as needed.

It ensures the application is usable on various devices, including desktops, tablets, and smartphones.


Future Enhancement


It plans to incorporate more detailed and customizable reporting options, including graphical representations of data.

Implementation of an automated notification system to alert students and staff about important updates, deadlines, and events.

Potential integration with other educational tools and platforms to enhance the functionality and user experience.

Bingo Game

Project Overview


The Bingo Game is an interactive and engaging web-based game developed using HTML, CSS, and JavaScript. This project focuses on providing a fun and dynamic user experience with classic Bingo gameplay, enhanced with modern web technologies.


Technologies used


HTML

CSS

JavaScript


Features


It provides a fully interactive Bingo game experience, where users can play Bingo in real-time with a digital Bingo sheet.

It implements a robust random number generator to ensure fair and unpredictable game outcomes.

It ensures the game is playable on various devices, including desktops, tablets, and smartphones, with a responsive and adaptive layout.

The game interface updates in real-time as numbers are called and marked on the Bingo card, providing a seamless user experience.

IT includes visual and audio feedback for game actions, such as marking numbers and winning, to enhance user engagement.

It supports multiple winning patterns, including traditional Bingo (horizontal, vertical, diagonal) and custom patterns.

It is designed with a user-friendly interface that makes it easy for players of all ages to understand and enjoy the game.


Current Status


The core gameplay mechanics are implemented and functional. The project is ongoing, with continuous improvements and feature additions.


Future Enhancement


Plans to add multiplayer functionality, allowing users to play Bingo with friends online.

Integration of a leaderboard system to track high scores and player rankings.

Development of themed Bingo cards and game modes for added variety and fun.
